# API Reference: HAG Novel Learning Mechanisms (Build 2.1)

This document details the classes and protocols introduced in Build 2.1 for the Holographic AI Governor system.

## 1. Holographic Weight-Encoding (`src.agents.holographic_memory`)

### `HolographicWeightEncoder`
**Description:** Encodes neural weights into a frequency manifold to prevent information loss.
- `encode(weights: torch.Tensor)`: Transforms weights to Fourier space.
- `decode(holographic_weights: torch.Tensor)`: Recovers weights from the distributed manifold.
- `recover_weights(weights: torch.Tensor, erasure_ratio: float)`: Full recovery cycle from partial weight erasure.

### `HolographicLayer(input_dim, output_dim)`
**Description:** A neural layer with built-in holographic protection. Supports `damaged` inference mode.

## 2. Symmetry-Aware Learning (`src.agents.lie_augmenter`)

### `LieAugmenter(input_dim, num_generators)`
**Description:** Discovers continuous algebraic symmetries in high-dimensional data.
- `forward(x)`: Applies an infinitesimal transformation generated by learned Lie Algebra generators.
- `get_invariance_loss(x)`: Measures how invariant the model's features are under the discovered symmetry groups.

## 3. Recursive Diffusion Reasoning (`src.agents.diffusion_reasoning`)

### `RecursiveDiffusionReasoning(state_dim, num_steps)`
**Description:** Implements the solution "crystallization" protocol.
- `crystallize(initial_state)`: Iteratively refines a latent solution vector using a diffusion refiner network.
- `solve_with_diffusion(query_vec, context_vec)`: Full protocol for synthesizing answers from multi-source embeddings.

## 4. Resilient Modeling (`src.agents.resilient_model`)

### `ResilientHAGModel(input_dim, hidden_dim)`
**Description:** The unified Build 2.1 model.
- Integrates `LieAugmenter` (Input) and `HolographicLayer` (Processing).
- `get_resilience_report(x)`: Evaluates recovery precision and symmetry invariance in a single report.

## 5. Natively Recursive Agent (`src.agents.native_recursive`)

### `NativelyRecursiveAgent(base_model, governor)`
**Description:** The primary interface for HAG interactions.
- Orchestrates recursive exploration in an isolated `NativeSandbox`.
- Performs integrity checks using the `HolographicGovernor` (EKRLS) at each step.
- Uses `RecursiveDiffusionReasoning` for the final crystallization of the answer.
